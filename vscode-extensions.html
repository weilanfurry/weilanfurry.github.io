VS Code 生产力扩展配置指南

📅 2025-10-08 ⏱️ 阅读时间：15分钟 🏷️ 开发工具

🎯 核心扩展推荐

1. GitLens - Git 增强工具

GitLens 增强了 VS Code 内置的 Git 功能，提供了丰富的代码注释和 Git 历史信息。

// 安装命令
ext install eamodio.gitlens

主要功能：

  • • 代码行级别的 Git 注释
  • • 文件历史记录和比较
  • • 分支管理和可视化
  • • 提交信息模板

2. Prettier - 代码格式化

Prettier 是一个固执己见的代码格式化工具，支持多种语言，确保代码风格一致。

// 安装命令
ext install esbenp.prettier-vscode

配置示例：

{
  "prettier.semi": true,
  "prettier.singleQuote": true,
  "prettier.tabWidth": 2,
  "prettier.trailingComma": "es5"
}

3. ESLint - JavaScript 代码检查

ESLint 用于识别和报告 JavaScript 代码中的问题，帮助维护代码质量。

// 安装命令
ext install dbaeumer.vscode-eslint

推荐配置：

{
  "eslint.validate": [
    "javascript",
    "javascriptreact",
    "typescript",
    "typescriptreact"
  ],
  "eslint.format.enable": true
}

4. Live Server - 实时预览

为静态和动态页面启动具有实时重载功能的开发本地服务器。

// 安装命令
ext install ritwickdey.liveserver

使用技巧：

  • • 右键 HTML 文件选择 "Open with Live Server"
  • • 自动检测文件更改并刷新页面
  • • 支持多浏览器同步

⚙️ 工作区配置

推荐的 settings.json 配置

{
  // 编辑器设置
  "editor.fontSize": 14,
  "editor.lineHeight": 1.5,
  "editor.tabSize": 2,
  "editor.wordWrap": "on",

  // 文件设置
  "files.autoSave": "onFocusChange",
  "files.trimTrailingWhitespace": true,

  // 格式化设置
  "editor.formatOnSave": true,
  "editor.defaultFormatter": "esbenp.prettier-vscode",

  // Git 设置
  "git.autofetch": true,
  "git.confirmSync": false
}

🎨 主题和图标

推荐主题

  • • One Dark Pro - 经典的暗色主题
  • • GitHub Theme - GitHub 风格主题
  • • Material Theme - Material Design 风格
  • • Night Owl - 专为夜间编码设计

文件图标主题

  • • vscode-icons - 丰富的文件图标
  • • Material Icon Theme - Material Design 图标
  • • Simple Icons - 简洁的图标集

🚀 高级技巧

1. 多光标编辑

使用 Ctrl/Cmd + D 选择下一个相同文本， Alt + 点击 添加多个光标。

2. 命令面板

使用 Ctrl/Cmd + Shift + P 打开命令面板， 快速执行各种操作。

3. 代码片段

创建自定义代码片段，提高编码效率。在设置中搜索 "snippets" 进行配置。

💡 总结

合理配置 VS Code 扩展可以显著提升开发效率。建议根据项目需求选择必要的扩展， 避免安装过多扩展影响性能。定期更新扩展和 VS Code 本身，享受最新的功能和优化。
